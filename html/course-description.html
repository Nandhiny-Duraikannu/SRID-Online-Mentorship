Cloud Computing

This course gives students an overview of Cloud Computing, which is the delivery of computing as a service over a network. The course will introduce the cloud and cover the topics of data centers, software defined networks and storage, cloud storage. We will start by discussing the clouds motivating factors, benefits, challenges, SLAs. We will describe several concepts behind data center design and management, which enable the economic and technological benefits of the cloud paradigm.
